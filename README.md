# ChefDrop

ChefDrop est une application iOS que j’ai développée en SwiftUI.  
Elle sert à transformer une conversation d’IA contenant une recette en une fiche de recette propre, structurée et utilisable.

## Fonctionnalités

- Import d’une conversation provenant d’un modèle d’IA (ChatGPT, Copilot, Claude, etc.)
- Extraction automatique du titre, des ingrédients, des étapes et des informations utiles
- Structuration propre de la recette dans une interface simple
- Calcul automatique des calories et des macros
- Génération d’une image de la recette à partir d’un prompt
- Stockage des images via Supabase Storage
- Interface moderne en SwiftUI
- Synchronisation iPhone/iPad via CloudKit
- Système de crédits : 5 recettes gratuites, puis achat de crédits supplémentaires

## Technologies utilisées

- SwiftUI
- Xcode
- iOS 17+
- Architecture MVVM
- CloudKit
- Supabase Storage
- Modèles d’IA pour la génération d’images
- Copilot / IA pour accélérer certaines parties du développement

## Installation

1. Cloner le dépôt
2. Ouvrir `ChefDrop.xcodeproj` dans Xcode
3. Compiler sur iPhone ou simulateur
